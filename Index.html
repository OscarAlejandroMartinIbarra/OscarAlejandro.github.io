Sistema de Monitoreo y Control Automatizado para Horno Industrial

Introducción

La automatización industrial ha revolucionado el monitoreo y control de procesos productivos, especialmente en sistemas térmicos como hornos. Este proyecto implementa un sistema con sensores y actuadores para garantizar eficiencia, seguridad y control en tiempo real mediante una interfaz en LabVIEW y una tarjeta Arduino.

Objetivo

Desarrollar e implementar un sistema de monitoreo y control automatizado para un horno industrial utilizando LabVIEW y Arduino, integrando sensores (gas, temperatura, flama y presencia) y actuadores (LEDs, buzzer, servomotor) para mejorar la seguridad y eficiencia operativa.

Automatización Industrial y Control de Procesos

La automatización reduce la intervención humana mediante tecnologías de monitoreo y control. Esto permite operar hornos industriales de forma más segura y eficiente.

LabVIEW

LabVIEW es un entorno de programación gráfica que permite crear interfaces y diagramas para controlar sistemas mediante adquisición de datos.

Comunicación Serial y LINX

LabVIEW se comunica con Arduino mediante el protocolo VISA y el toolkit LINX, permitiendo la lectura y escritura de datos en tiempo real.

Sensores Utilizados

  • Sensor de gas (analógico)
  • Sensor de temperatura (analógico)
  • Sensor de flama (digital)
  • Sensor de presencia (digital)

Actuadores

  • LEDs indicadores
  • Buzzer de alerta
  • Servomotor (control de puerta)

Desarrollo del Proyecto

El sistema se comunica con Arduino usando VISA. Se usan bloques de LabVIEW como Analog Read, Digital Read/Write y PWM Write. Los sensores monitorean condiciones críticas y los actuadores responden con alarmas o mecanismos físicos.

Bloques clave en LabVIEW

  • Serial Port (VISA): comunicación con Arduino
  • Analog Read: lectura de sensores de gas y temperatura
  • Digital Read: lectura de sensores de flama y presencia
  • Digital Write: control de LEDs y buzzer
  • PWM Write: control del servomotor de la puerta

Funcionamiento del Sistema

El sistema opera en tiempo real. Los sensores proporcionan datos continuamente, que son analizados y comparados con umbrales críticos para activar respuestas lógicas y físicas.

  • Temperatura > 125 °C: activa alarma y bloquea la puerta
  • Gas sin flama: se activa alarma por posible fuga
  • Presencia: activa advertencia de seguridad

Casos de Uso y Seguridad

El sistema responde ante distintos escenarios:

  • Alta temperatura: activa LED de advertencia y bloquea la puerta
  • Detección de flama: asegura la presencia de combustión
  • Fuga de gas: sin flama activa el buzzer y la advertencia
  • Presencia de persona: bloquea el sistema si hay riesgo

La interfaz en LabVIEW simula gráficamente cada situación y muestra animaciones para una mejor comprensión del estado del horno.
